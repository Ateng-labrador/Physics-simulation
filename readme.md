## ðŸ’» Module ðŸ’»

NumPy adalah modul Python untuk komputasi numerik yang menyediakan array multidimensi cepat (ndarray), serta fungsi matematika seperti aljabar linear, statistik, dan operasi vektor. NumPy membuat perhitungan skala besar menjadi efisien dan sering digunakan dalam sains data dan fisika.

Matplotlib adalah modul Python untuk membuat grafik dan visualisasi data. Dengan antarmuka sederhana seperti MATLAB, Matplotlib memungkinkan pembuatan plot garis, scatter, histogram, dan visualisasi lainnya, serta mudah digabungkan dengan NumPy dan Pandas.
