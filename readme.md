NumPy (Numerical Python) adalah modul fundamental dalam ekosistem Python untuk komputasi ilmiah. NumPy menyediakan struktur data utama bernama ndarray, yaitu array multidimensi yang sangat efisien dan jauh lebih cepat dibandingkan list Python biasa. Dengan NumPy, operasi matematika seperti aljabar linear, transformasi Fourier, statistik, dan manipulasi data skala besar dapat dilakukan dengan mudah dan cepat karena fungsi-fungsinya ditulis menggunakan optimasi C di belakang layar. NumPy juga mendukung broadcasting, yaitu kemampuan melakukan operasi antar array dengan ukuran berbeda tanpa perlu melakukan looping manual, sehingga sangat berguna untuk simulasi, pemrosesan sinyal, data science, serta aplikasi fisika dan teknik.

Matplotlib adalah modul Python yang digunakan untuk membuat visualisasi data dengan fleksibilitas sangat tinggi. Library ini memungkinkan pengguna membuat berbagai jenis grafik, seperti line plot, scatter plot, histogram, bar chart, hingga visualisasi lebih kompleks seperti animasi dan subplot grid. Di bawah modul Pyplot, pengguna dapat membuat grafik dengan sintaks yang sederhana layaknya MATLAB, sehingga sangat cocok untuk riset, analisis data, maupun presentasi ilmiah. Matplotlib juga dapat dikombinasikan dengan NumPy, Pandas, dan SciPy sehingga memudahkan visualisasi dari dataset besar atau hasil komputasi numerik. Kemampuan untuk mengatur warna, label, gaya garis, anotasi, dan layout menjadikan Matplotlib sebagai pilihan utama untuk plotting ilmiah di Python.
